Projects

Resume Builder Application

The Resume Builder Application is a web-based tool designed to help users create, manage, and search resumes. Leveraging the power of HTML, CSS, and JavaScript for the front-end, and utilizing JSON Server as a mock back-end, this project offers a seamless experience for building and searching resumes. Users can input their resume details, which are then stored and managed by JSON Server, providing a realistic API experience.

Todoslist

Todoslist is a simple, yet powerful to-do list application designed to help users manage their daily tasks effectively. Built using HTML, CSS, and JavaScript, this application provides an intuitive interface for adding, updating, listing, deleting, and marking tasks as complete or incomplete. The tasks are stored in the browser's local storage, ensuring data persistence across sessions without the need for a back-end server.
